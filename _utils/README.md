## fontefideliti.ru utils

Utilities used to simplify content creation for https://github.com/arhad/fontefideliti.ru.

### Files

<dl>
	<dt>image-resizer-*.bat</dt>
	<dd>[ImageMagick](https://www.imagemagick.org)-based utilities that:
		<ol>
			<li>Take images passed as command line parameters;</li>
			<li>Create two versions of each image: a *large one* that fits into some rectangle and a *preview* that has a fixed height (specified in a script name instead of an asterisk);</li>
			<li>Losslessly reduces size of the images using [FileOptimizer](https://nikkhokkho.sourceforge.io/static.php?page=FileOptimizer).</li>
		</ol>
	</dd>
</dl>
